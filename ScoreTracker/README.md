## 📖 Problem Statement
This is a web-based Student Score Tracker application where users can enter subject-wise scores through an HTML form. The application stores the data using HTTP Session and calculates the average score dynamically using Java Servlets. The results are displayed in a tabular format on the browser.

---

## 🛠️ Technologies Used
- Java (Servlets)
- HTML
- CSS (basic)
- Apache Tomcat 10
- Eclipse IDE

---

## ⚙️ How to Run This Project
1. Import project into Eclipse as Dynamic Web Project  
2. Configure Apache Tomcat server (Tomcat 10)  
3. Run project on server  
4. Open browser and go to:  
   http://localhost:8080/YourProjectName/index.html

---

## 📊 Output
- Input form to enter subject and score  
- Display table of all entered scores  
- Display calculated average score  
- Reset option to clear session data  

---

## 💡 Concepts Used
- HttpServlet (ScoreServlet)   
- HttpSession for data storage  
- Request and Response handling  
- Form handling using HTML + Servlet integration
